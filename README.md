# Online Phone Store
## Technology
Java, JSP, SQL Server, Bootstrap.

## This Website let you:
1) Login with google
2) Receive email to be the first to know about store news
3) Interact with chatbot or customer service
4) Checkout

6) httpcore-4.4.9
